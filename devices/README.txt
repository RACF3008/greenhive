-----------------------------------------------------
|   MICROSERVICIO DE ADMINISTRACIÓN DE DISPOSITIVOS
----------------------------------------------------

Función:
- Registrar nuevos dispositivos en la API y asignarles un ID
- Guardar y publicar información del dispositivo, como:
    - Tipo de dispositivo
    - Nombre asignado al dispositivo
    - IP del gateway (RPi)
    - Status (online/offline/maintainance)
    - ID del usuario asociado
    - Última acutlización del dispositivo
- Notificar de la desconexión de algún dispositivo
- Validación de dispositivos antes de interactuar con la API
